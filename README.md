## Deploy
### GitHub Pages
- Set `homepage` in package.json to `https://<username>.github.io/<repo-name>/` (optional but recommended).
- Run:
```bash
npm run deploy
```

### Vercel
- Push this repo to GitHub, then import it on vercel.com and click Deploy.
